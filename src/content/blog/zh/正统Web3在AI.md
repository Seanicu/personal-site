2001年，Tim Berners-Lee 提出“语义网”，核心理念是计算机能通过网页元数据读懂网页内容；后来一些技术极客补充了个人数据去中心化的理念，即用户的数据由大互联网公司独家控制，而是由用户自己掌握数据内容、数据权限。这便是最早的、最正统的 web 3。在这样的构想下，每个人都可以把自己的数据与各种网页服务相结合，满足个性化的需求。不过后来的事大家都知道了，那个加密货币的 Web 3 篡位了。

而现在，AI 强大的自然语言处理能力，正在重构一切。一个现象是，即便从未接触过代码的小白，也能够借助智能体，快速搭建出属于自己的应用。这不正是曾经 Web 3 的构想？

当用户需要执行一个复杂的数据处理流程、创建一个自己的投票网站、发布个人作品集，都是一句话的事。

传统程序员的生存空间被极大压缩，甚至被取代了。

“让信息像自来水一样流动，让 Tokens 像电力一样普及”。
