## 博客网站首页 (Article Detail Page) 设计需求文档

### 1. 整体视觉风格 (Visual Identity)

- **设计哲学：** 极简主义、文学感、瑞士排版风格（Swiss Design）的变体。
- **背景色（纸张感）：** 使用极浅的灰黄色或灰绿色，模拟高质感艺术纸张。
  - *推荐色号：* `#F5F5F0` (暖灰) 或 `#F2F2EB` (浅灰绿)。
- **文字基础色：** 非纯黑，建议使用深炭灰色 `#1A1A1A` 以增加阅读舒适度。

### 2. 布局结构 (Layout)

- **顶部导航 (Header)：** 左侧网站名，居中为导航菜单（Blogs, Works, About）。
- **主体内容 (Main Content)：**
  - **分栏布局逻辑：** 采用非对称布局。左侧较窄部分（约 20-30% 宽度）放置栏目名称（如 "About this blog"、"Blog posts"）；右侧较宽部分（约 70-80% 宽度）放置具体内容（如博客文章列表）。
- **间距 (Spacing)：** 留白大胆，行间距（Line Height）保持在 1.6 - 1.8 之间，确保视觉呼吸感。

### 3. 字体系统 (Typography)

> **核心要求：全站使用衬线体（Serif）。**

- **英文：** 使用 `Libre Baskerville`。
- **中文：** 使用 `思源宋体 (Source Han Serif SC)` / `Noto Serif SC`。
- **字阶规则：**
  - **大标题 (Hero)：** 48px - 64px, Bold。
  - **内容标题 (Post Title)：** 24px - 28px, Medium。
  - **栏目名 (Label)：** 16px, Bold。
  - **正文/日期：** 16px - 18px，日期颜色减淡（如 `#888888`）。

### 4. 核心交互动效 (Hover Interaction)

> **核心要求：瞬间触发的“大色块”填充效果。**

- **触发对象：** 所有可点击的链接、导航项、文章列表项。
- **静态样式：** 文字下无下划线，保持简洁。
- **悬浮态 (Hover State)：**
  - **背景变化：** 整个（父）元素的大矩形区域瞬间填充为鲜艳的色块（推荐橙色 `#FF4500` 或青蓝色 `#0055FF`）。
  - **文字变化：** 文字颜色瞬间变为白色 `#FFFFFF`。
  - **动画速度：** `transition: 0s` 或 `0.1s` (Linear)，追求“啪”一下跳出来的果断感，严禁缓慢淡入。

### 5. 元素细节描述表

| **元素**       | **描述**                                                   |
| -------------- | ---------------------------------------------------------- |
| **背景**       | 全局背景色 `#F5F5F0`，无边框，无投影。                     |
| **文章列表**   | 纵向排列，上方标题，下方日期。日期与标题间距约 8px。       |
| **交互区域**   | 鼠标悬停在标题或日期所在行时，整个行（或标题矩形框）变色。 |
| **分隔线**     | 尽量不使用实线，通过留白（Margin）来进行视觉分割。         |
| **移动端适配** | 左右分栏在手机端改为上下堆叠，保持大字号和衬线体。         |
