## 博客文章详情页 (Article Detail Page) 设计需求文档

### 1. 布局结构 (Detail Layout)

- **延续非对称比例：** 保持左侧 20-30% 栏位，右侧 70-80% 主体内容的分配。
- **左侧栏 (Meta Information)：** * 不再显示“About this blog”，而是显示当前文章的元数据：**发布日期、预计阅读时间、分类/标签**。
  - 采用固定定位（Sticky），当用户向下滚动阅读时，这些信息可以停留在视野内，作为视觉锚点。
- **右侧主体 (Article Body)：**
  - **文章标题 (H1)：** 位于右侧顶部，字号与首页 Hero 标题一致（48px-64px），加粗衬线体。
  - **正文容器：** 设置最大宽度（Max-width, 如 720px - 850px），防止长难句跨度过大导致阅读疲劳。
  - **留白：** 标题与正文之间预留极大的纵向空间（120px+）。

### 2. 正文排版系统 (Typography Typography)

> **核心：模拟纸质书籍的版心感。**

- **段落 (Body Text)：** * 字号：18px - 20px（略大于首页，适合长文本）。
  - 行高：1.8 - 2.0（确保行间有足够的空气感）。
  - 首字下沉 (Drop Cap)：每篇文章的第一段第一个字可以设置为巨大的衬线体加粗，占据 3 行高度，增强文学感。
- **层级标题 (H2, H3)：**
  - **H2：** 32px，衬线加粗，上方留白 2em，下方留白 1em。
  - **H3：** 24px，衬线加粗。
- **引用 (Blockquote)：**
  - 不使用侧边线。改为文字整体向右缩进，并使用斜体（Italic）衬线体，字号略微加大或缩小，形成独特的灰度感。

### 3. 多媒体与附件 (Media & Details)

- **图片 (Images)：** * 图片宽度可以超出正文容器，达到 100% 宽度或悬出效果（Bleed Layout）。
  - 图片下方配有 14px 的衬线体注释（Caption），颜色为浅灰色 `#888888`。
- **脚注 (Footnotes)：** * 正文中的角标点击后，页面瞬间跳转至底部，或在左侧窄栏显示。

### 4. 详情页特有的交互 (Detail Interaction)

> **核心：将“大色块”逻辑引入阅读场景。**

- **正文内链接 (In-text Links)：** * 静态下带有极细的下划线（或者无下划线，仅区分颜色）。
  - **Hover 时：** 链接文字背景瞬间变为**橙色 `#FF4500`**，文字变白。色块应具有微小的内边距，看起来像是一个醒目的标签。
- **底部导航 (Post Navigation)：** * 文章末尾显示“上一篇”和“下一篇”。
  - 每个导航块为一个巨大的矩形。鼠标悬停时，整个矩形区域瞬间填充为色块，这是一种强烈的暗示，引导用户继续阅读。
- **返回按钮 (Back to Home)：** * 左上角放置一个简单的 “← Back”，悬停时文字被矩形色块包围。

### 5. 文章页元素细节表

| **元素**       | **描述**                                                     |
| -------------- | ------------------------------------------------------------ |
| **标题 (H1)**  | `Libre Baskerville` / `思源宋体`, 64px, 炭灰色。             |
| **左侧元数据** | 纵向排列。Label 为 14px Bold，数据值为 14px Regular。        |
| **正文链接**   | Hover 瞬间：`Background: #FF4500; Color: #FFF; transition: 0s;` |
| **图片展示**   | 允许“超宽”显示，打破垂直对齐线，增加版面灵动性。             |
| **阅读进度条** | （可选）页面顶部一个极细的 2px 进度条，使用橙色，随滚动而增长。 |
